## What is Mint?

Mint is a Windows playbook that does what most people wish Windows did by default: removes the junk, locks down the privacy leaks, and gets out of your way.

It's not a script that blindly disables everything it can find. It's a carefully considered overhaul that strips telemetry, kills unnecessary services, and debloats the OS — while keeping things stable enough for everyday use. Whether you're gaming competitively, running a professional workstation, or just tired of Microsoft collecting your data, Mint gives you a foundation worth building on.

---

## The Three Things Mint Gets Right

**Performance** — Background processes and non-essential services are disabled at the source. Less CPU overhead, less RAM waste, more of your hardware dedicated to what you actually care about.

**Privacy** — Microsoft's data collection runs deep. Mint goes deeper. Registry modifications, host-level blocks, and systematic dismantling of telemetry pipelines mean your data stays on your machine.

**Security** — Most optimization playbooks sacrifice security for speed. Mint doesn't. Attack Surface Reduction rules, exploit protection, and hardened authentication policies are all included — because a fast system that's also compromised isn't much use to anyone.

## Features

### A Cleaner Desktop
The taskbar is stripped back to the essentials: centered, small icons, auto-hide enabled, no clock, no notification center, no tray clutter. File Explorer runs in Compact Mode with file extensions always visible and the Home/Gallery sidebar removed. Desktop icons are hidden by default. It sounds minimal because it is.

### Thorough Debloating
Microsoft Edge, OneDrive, Teams, and most preinstalled UWP apps are removed properly — not just hidden. The classic Windows 10 right-click context menu is restored, and power-user additions like "Take Ownership" and "Open with Notepad" are included out of the box.

### Fine-Tuned Services
Error reporting, telemetry daemons, and background maintenance tasks are selectively disabled. Not everything — just the things that waste resources without benefiting you.

### Windows Enterprise LTSC Conversion
Mint can perform a genuine edition conversion to Windows Enterprise LTSC, giving you the most stable, long-term-supported version of Windows available. Less bloat, longer update cycles, consistent experience.

## Presets

Mint ships with five profiles. Pick the one that matches your needs — you choose during setup.

| Preset | What it does |
|---|---|
| **Balanced** | A sensible everyday setup. Fast, functional, and stable without going overboard. |
| **Gaming Focused** | Low latency, high FPS. Game processes are prioritized, network throttling is removed. |
| **Performance Focused** | Aggressive resource reduction. Every background cycle is accounted for. |
| **Privacy Focused** | All telemetry hosts blocked. Cloud-integrated features disabled. Minimal footprint. |
| **Paranoid Mode** | Maximum aggression across the board. Every block, every lock, fully applied. |

## Installation

> **Before you start:** A clean Windows install is strongly recommended. Create a system restore point — just in case.

1. **Download AME Wizard** from [ameliorated.io](https://ameliorated.io)
2. **Download Mint** — grab the latest `.apbx` from the [Releases page](https://github.com/Mint-D8/Mint-Playbook/releases)
3. **Import the playbook** into AME Wizard
4. **Enter the decryption password** when prompted: `malte`
5. **Choose your preset** and any optional components
6. **Restart** when the wizard finishes

That's it. Welcome to a cleaner Windows.

### Supported Versions

- Windows 10 (22H2)
- Windows 11 (22H2 / 23H2 / 24H2 / 25H2)
